PA County Migration
An investigation into Pennsylvania's migration patterns using IRS data.

This project examines county-to-county migration in Pennsylvania by analyzing the 2017-2018 IRS migration dataset, which can be found on the IRS website.

Data was retrieved as a .csv file, containing a row for each county-to-county flow. Some flows were aggregated (Total Migration-US and Foreign) while some reflected a single county to another county. Specific flows with fewer than 20 returns or exemptions are obscured from this dataset for privacy.

An R script was written to do the following:

  1. Import and clean the data
  2. Write to a new table in a SQLite database
  3. Pivot the data so rows reflect individual Pennsylvania counties
  4. Join the table to a county shapefile in preparation for visualization

Maps were designed in ArcGIS Pro to show the percent of migrants in 2018 who had moved to each Pennsylvania county from a different state.
